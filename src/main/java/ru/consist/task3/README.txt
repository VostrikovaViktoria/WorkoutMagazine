Приложение - "Дневник тренировок"

Приложение предназначено для любителей спорта, которые хотят
записывать информацию о выполненных тренировках и отслеживать свой прогресс.

Спортсмен-любитель сможет добавлять такую информацию:
- дата тренировки;
- вид тренировки (кросс, силовая и т.д.);
- время тренировки:
- километры;
- описание тренировки;
- пульс (среднее значение);
- сожженые калории;
- самочувствие.

Будут реализованы возможности получения, добавления, удаления, изменения этой информации (CRUD),
получение информации о суммарном количестве преодоленных километров, о суммарном времени, затраченном на тренировки,
о суммарном количестве сожженых калорий (бизнес-методы).

Данные проекта будут выгружаться в файл *.xml
